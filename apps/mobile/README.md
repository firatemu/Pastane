# Pasta-Hane Mobile

Expo + React Native customer mobile app for the Pasta-Hane storefront.

## Run

```bash
pnpm install --filter @pastane/mobile
pnpm --filter @pastane/mobile start
```

Set `EXPO_PUBLIC_API_URL` before running on a real device.

Local API hints:

- Android emulator: `http://10.0.2.2:3003`
- iOS simulator: `http://localhost:3003`
- Physical device: host LAN IP, for example `http://192.168.x.x:3003`
