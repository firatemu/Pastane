# Git Workflow

## Branching
Use short-lived branches such as:
- `codex/phase-01-foundation`
- `feat/order-flow`
- `fix/payment-timeout`

## Commits
Follow Conventional Commits and keep each commit focused.

## Phase completion
Use an explicit completion commit once checks pass, for example:
`chore: complete phase 01 foundation`

Do not mix future-phase work into a current-phase branch.
